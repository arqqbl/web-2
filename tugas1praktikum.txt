jawaban

1. Dokumen root (document root) adalah direktori utama di server web yang digunakan untuk menyimpan file-file situs web yang dapat diakses oleh pengguna melalui browser. Semua permintaan HTTP dari pengguna akan diarahkan ke direktori ini, kecuali dikonfigurasi sebaliknya.
2. Tidak harus. Lokasi default dokumen root untuk Apache pada XAMPP memang berada di C:\xampp\htdocs, tetapi bisa diubah dengan mengedit file konfigurasi Apache (httpd.conf).
3.git init → Menginisialisasi repository Git baru dalam direktori saat ini.
git clone <URL> → Menyalin repository yang sudah ada dari URL ke komputer lokal.
git remote → Mengelola daftar repository jarak jauh (remote). Digunakan untuk menambah, menghapus, atau melihat remote repository.
git add <file> → Menambahkan file ke staging area sebelum dilakukan commit.
git commit -m "pesan" → Menyimpan perubahan yang telah ditambahkan ke staging area ke dalam repository lokal dengan pesan deskriptif.
git push → Mengirim perubahan dari repository lokal ke repository remote (misalnya GitHub atau GitLab).
git checkout <branch> → Beralih ke branch lain dalam repository. Juga bisa digunakan untuk berpindah ke commit tertentu.
4. Git Hub = Fokus pada kolaborasi terbuka dan integrasi dengan banyak alat lain,Harus menggunakan layanan tambahan (GitHub Actions), Tidak tersedia secara langsung (hanya menggunakan GitHub Enterprise)
   Git Lab = Lebih menekankan kontrol penuh dengan opsi self-hosted, Sudah terintegrasi secara bawaan, Bisa di-host secara pribadi di server sendiri
5. Variabel User: Hanya berlaku untuk akun pengguna tertentu dalam sistem. Setiap pengguna dapat memiliki variabel lingkungan yang berbeda.
Variabel System: Berlaku untuk semua pengguna dalam sistem. Biasanya digunakan untuk menentukan pengaturan global, seperti PATH sistem atau lokasi program tertentu.
6. Keduanya menyimpan nilai yang dapat digunakan dalam program atau sistem operasi. Perbedaannya:

Variabel System bisa diubah oleh administrator sistem dan mempengaruhi seluruh sistem.
Variabel Constant biasanya bersifat tetap dalam kode program dan tidak bisa diubah setelah dideklarasikan.
